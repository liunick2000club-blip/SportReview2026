# SportReview 应用开发路线图 (Roadmap)

本项目致力于打造一款高度个性化的运动记录与财务分析应用。以下是当前已确认的进阶开发方向：

## 🚀 第一阶段：数据可视化与趋势分析 (Trends)
*   **周/月跑步里程趋势：** 使用折线图展示每周或每月的累计跑步距离，直观查看耐力进步。
*   **攀岩频率分布图：** 优化柱状图，支持按月份筛选，对比不同月份的打卡热情。
*   **运动类型占比：** 增加饼图，展示跑步、骑行、攀岩、其他训练的时间/次数分配比例。

## 🧗‍♂️ 第二阶段：攀岩馆深度统计模块 (Gym Analysis)
*   **单馆详情页：** 点击馆名进入详情。
    *   统计在该馆的总次数、总时长（如果未来记录时长的话）、总花费。
    *   **性价比分析：** 计算该馆的单次平均成本（单次票 vs 年卡分摊）。
    *   **训练回顾：** 聚合展示在该馆留下的所有心得体会。

## 🎯 第三阶段：目标管理与激励系统 (Goals)
*   **月度目标设定：** 允许设置本月的跑步里程（如 50km）、攀岩次数（如 12次）。
*   **首页进度条：** 在仪表盘实时显示目标达成百分比。
*   **连续打卡记录：** 统计并展示最长连续运动天数，增加记录动力。

## 💾 第四阶段：数据主权与安全 (Data & Safety)
*   **一键导出功能：** 支持将数据库记录导出为 CSV 或 Excel 文件，方便备份。
*   **数据库快照：** 自动或手动备份 `dev.db` 镜像。
*   **多端适配优化：** 进一步优化移动端布局，确保在手机端录入和查看图表更加顺滑。
